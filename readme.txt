#How To Use WebCrawler

    1. DOWNLOAD AND EXTRACT THE ZIP FILE
    2. TO ACTIVATE VIRTUAL ENVIRONMENT
	    For mac/linux: . venv/bin/activate
	    For Windows: venv\Scripts\activate
    3. RUN THE APP:
    	Open the downloaded folder in the terminal and run: python3 app.py
    4. TO SCAN:
    	Open link: http://127.0.0.1:8000/ in your web browser and enter the desired url and scan.
    	
    	You Report will be ready in seconds!
